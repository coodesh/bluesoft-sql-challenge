# Bluesoft SQL Challenge

## Introdução


Este é um teste para que possamos ver as suas habilidades em migração de dados.

Neste teste queremos que você criei um cenário de migração de dados.

[SPOILER] As instruções de entrega e apresentação do challenge estão no final deste Readme (=

### Antes de começar
 
- Considere como deadline da avaliação a partir do início do teste. Caso tenha sido convidado a realizar o teste e não seja possível concluir dentro deste período, avise a pessoa que o convidou para receber instruções sobre o que fazer.
- Documentar todo o processo de investigação para o desenvolvimento da atividade (README.md no seu repositório); os resultados destas tarefas são tão importantes do que o seu processo de pensamento e decisões à medida que as completa, por isso tente documentar e apresentar os seus hipóteses e decisões na medida do possível.


## **Migração de dados**

Queremos que você criei um cenário de migração de dados.
Com os principais comandos. Como você faria?
Utilizando esses comandos SQL, com uma base de banco de dados de sua preferência.

Comandos:

- inserção
- insert
